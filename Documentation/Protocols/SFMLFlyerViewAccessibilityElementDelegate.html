public protocol SFMLFlyerViewAccessibilityElementDelegate: NSObjectProtocol

The SFMLFlyerViewAccessibilityElementDelegate defines methods that allow further customization of accessiblity elements found in the SFMLView.

  • accessibilityValueAddendum(itemID:)

    This method should be implemented if additional text should be added to the end of the constructed accessibilityLabel. The string returned will be appended to the value of accessibilityLabel with a period separating them for a pause in VoiceOver mode.
