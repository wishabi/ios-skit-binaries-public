SFMLViewDelegate

@objc
public protocol SFMLViewDelegate

The SFMLViewDelegate protocol defines methods that allow you to handle certain SFMLView events, such as when an item is tapped or long-pressed.

It also forwards some UIScrollViewDelegate methods in case they are needed.

  • sfmlViewVisibilityChanged(_:isVisible:)

    Calls the delegate when the view’s visibility has changed. Visibility is dependent on the isVisible property being set by the user. It is not automatically computed. Internally, other objects (such as analytics objects) listen for visibility events.

    Declaration

    Swift

    func sfmlViewVisibilityChanged(_ view: SFMLView, isVisible: Bool)

    Parameters

    view

    The SFMLView whose visibility changed.
    isVisible

    Whether the view is visible or not. true is only returned if the has been loaded (isLoaded) and is visible `isVisible). Both are public properties of the view.
