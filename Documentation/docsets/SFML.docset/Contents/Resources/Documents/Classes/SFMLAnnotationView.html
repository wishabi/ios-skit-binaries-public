SFMLAnnotationView

@objc
open class SFMLAnnotationView : UIView

Annotation views representing SFMLAnnotations should be subclasses of this view. Animations for insertion/removal can be implemented if desired.

  • insertAnnotationView(animated:)

    Called when this annotation view is added. Subclass can override.
