SFMLAnalytics

@objc
public class SFMLAnalytics : NSObject
extension SFMLAnalytics: SFMLViewDelegate
extension SFMLAnalytics: SFMLViewWayfinderDelegate

A configuration and convenience class for handling analytics in the framework. This class implements SFMLViewDelegate and SFMLViewWayfinderDelegate and has internal timers and logic to decide when to trigger SFMLViewAnalyticsDelegate events.

SFMLViewWayfinderDelegate

  • storefrontSFMLViewWayfinderWillAppear(_:)

    Note on this implementation:

    When the wayfinder quick scroll is being used, it is setting the content offset manually and the initial setting makes the scrollView call didEndDecelerating which in turn starts tracking item impressions. We stop the impressions timer so if the user starts using the wayfinder quick scroll (which opens the wayfinder list), it stops any impressions timer in progress which if fired would have erroneously fired an impressions beacon when there hasn’t been seconds of dwell time.

    If not using this implementation of analytics (i.e. not setting the delegate of the Analytics object), make sure to keep this in mind.
