SFMLAccessibilityElement

public class SFMLAccessibilityElement: UIAccessibilityElement

All accessiblity elements found in the SFMLView will be subclasses of SFMLAccessibilityElement.

  • frame

    The frame of this accessiblity element. This property should not be changed. It is only used when running on iOS 9 and lower. Its value is used to compute its accessibilityFrame whenever the scroll view scrolls.
